## Tasky označené ako automatizovateľné

- Zber veľkostí tričiek → Google Form → Master Sheet
- Check-in účastníkov → QR kód v Tootoot
- Sledovanie rozpočtu → Google Sheet s automatickými súčtami
- Sponzorský pipeline → CRM v Notion/Sheets
- Certifikáty cien → Mail merge šablóna v Canva/Google Slides
- Social media scheduling → Buffer/Later
- E-mailové kampane → Mailchimp automatické sekvencie
- Grafiky → Canva šablóny s branding kitom
- Oslovovanie škôl → Mail merge
- Tootoot → Google Sheet sync
- Discord setup → Šablóna servera
- Zber biografií mentorov → Google Form → Facilitátorský deck
- Počty diét → Automatický súhrn z registračného formulára
